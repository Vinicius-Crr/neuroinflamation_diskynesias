## Sobre o projeto

Este repositório contém todos os scripts utilizados na análise genética de discinesias induzidas por levodopa em pacientes com doença de Parkinson, utilizando dados do consórcio **AMP-PD** (Accelerating Medicines Partnership – Parkinson's Disease) e **GP2** (Global Parkinson's Genetics program)
